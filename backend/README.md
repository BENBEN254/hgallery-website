# HGALLERY - Backend API

## 🚀 Production-Ready E-Commerce Backend

### Tech Stack

- **Node.js** - Runtime Environment
- **Express.js** - Web Framework
- **MongoDB** - Database
- **Mongoose** - ODM
- **JWT** - Authentication
- **Cloudinary** - Image Upload
- **Nodemailer** - Email Service
- **Winston** - Logging

### Features

- ✅ User Authentication (Register, Login, JWT)
- ✅ Product Management (CRUD)
- ✅ Category Management
- ✅ Order Processing
- ✅ Shopping Cart
- ✅ Wishlist
- ✅ Blog Posts
- ✅ Service Management
- ✅ Project Portfolio
- ✅ Testimonials
- ✅ Image Upload (Cloudinary)
- ✅ Email Notifications
- ✅ Rate Limiting
- ✅ Security Headers
- ✅ Input Validation
- ✅ Error Handling
- ✅ Database Seeding

### Installation

```bash
# Clone repository
git clone https://github.com/hgallery/backend.git
cd backend

# Install dependencies
npm install

# Copy environment variables
cp .env.example .env

# Update .env with your values

# Run in development
npm run dev

# Run in production
npm start

# Seed database
npm run seed
```
